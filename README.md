# ✉️ Gmail整理ツール

このフォルダには2つのツールが入っています。**どちらか片方だけでも、両方でも使えます**(同じApps Scriptプロジェクトに同居できます)。

| ツール | ファイル | 向いている用途 |
|---|---|---|
| **メール仕分けカード**(おすすめ) | `Triage.gs` + `Triage.html` | 単語帳みたいにキー1発/1タップでパパっと人力仕分け |
| 自動整理シート | `Code.gs` | 設定に従ってメルマガ等を毎日自動でアーカイブ |

# 🃏 メール仕分けカード(Triage)

受信トレイのメールが**1枚ずつカードで表示**され、キー1発 or ボタン1タップで捌けるウェブアプリです。スマホでも使えます。

## 操作

| キー | 動き |
|---|---|
| `A` / `←` | アーカイブ(トレイから外す) |
| `S` | ★重要(★とラベルを付けてトレイに残す) |
| `D` | ゴミ箱(30日は復元可能) |
| `1`〜`8` | 自分で決めたラベルを付けてアーカイブ |
| `Ctrl`+`1`〜`8` | **複数ラベル選択**(選んでおいて、最後のラベルを普通に押す or `Enter`で全部まとめて付与) |
| `Enter` / `Esc` | 選択中ラベルの確定 / 解除 |
| `Space` / `→` | あとで(スキップ) |
| `Z` | 元に戻す |
| `Shift`+上記 | **同じ差出人のメールをまとめて同じ処理**(メルマガ一掃に便利) |

### スマホ操作

| 操作 | 動き |
|---|---|
| カードを**右スワイプ** | アーカイブ |
| カードを**左スワイプ** | あとで(スキップ) |
| ラベルを**長押し** | 複数ラベル選択(最後のラベルを普通にタップで確定) |

- 本文は3行まで表示され、長いメールは「…詳細を読む」で全文(先頭600字)に展開できます。
- 「⚙ ラベル編集」からボタンの内容を自由に変えられます(Gmail側にラベルが自動作成されます)。

### ホーム画面アプリ化について

- **iPhone**: SafariでウェブアプリURLを開く → 共有 → 「ホーム画面に追加」
- **Android**: Chromeでメニュー → 「ホーム画面に追加」
- アプリ用のメタタグ(テーマカラー等)は入れてありますが、Apps Scriptの仕組み上(ページがGoogleのラッパー内で動くため)**オフライン対応の完全なPWAインストールはできません**。ホーム画面から1タップで全画面起動、までは可能です。完全PWA化したい場合は画面側をGitHub Pagesに分離する構成に変えられるので、必要になったら相談してください。

## セットアップ(1回だけ)

1. <https://script.google.com> で新しいプロジェクトを作る(既にシート版を入れた人はそのプロジェクトでOK)
2. エディタ左の「ファイル +」→ **スクリプト** → 名前を `Triage` にして、`Triage.gs` の中身を貼り付け
3. 「ファイル +」→ **HTML** → 名前を `Triage` にして、`Triage.html` の中身を貼り付け → 保存
4. 右上「**デプロイ → 新しいデプロイ**」→ 歯車で「**ウェブアプリ**」を選択
   - 「アクセスできるユーザー」= **自分のみ**
   - 「デプロイ」→ 初回は権限の許可(自分のアカウント → 詳細 → 移動 → 許可)
5. 表示された**ウェブアプリのURLをブックマーク**(スマホのホーム画面に追加も便利)

以降はそのURLを開くだけ。溜まったメールを単語帳感覚で捌けます。

> 💡 貼り付ける前に操作感を試したい場合は、`Triage.html` をそのままブラウザで開くと**サンプルメールのデモモード**で動きます(実メールには触れません)。

# 📊 自動整理シート(旧版・併用可)

Gmailをスプレッドシートから直感的に整理するツールです。
**コードを触るのは最初の貼り付け1回だけ。** あとは全部シート上のメニューとセルで操作します。

## できること

| 困りごと | 動き |
|---|---|
| 重要メールが埋もれる | 指定した送信元・キーワードに ★＋「★重要」ラベル → トレイに残す |
| 受信トレイが多すぎる | メルマガ/SNS/通知/フォーラムをラベル付けしてトレイから外す |
| 定期的に自動整理したい | メニューでONにすると毎朝7時に自動実行 |

- **削除は一切しません**(トレイから外す=アーカイブ。検索すれば残っています)
- ★付き・「★重要」ラベルのメールは絶対に動かしません
- 実行前に「**プレビュー**」で対象メールを目で確認できます

## セットアップ(5分・1回だけ)

1. ブラウザで **[sheets.new](https://sheets.new)** を開く(新しいスプレッドシートが作られます)
   ※ 整理したいGmailのアカウントでログインしていること
2. シート名を「Gmail整理」などにしておく
3. メニュー **拡張機能 → Apps Script** をクリック
4. 開いたエディタの中身を全部消して、`Code.gs` の中身を**まるごと貼り付け** → 保存(Ctrl+S)
5. エディタは閉じて、**スプレッドシートに戻ってページを再読み込み**(F5)
6. 数秒待つと上部メニューに「**✉️ Gmail整理**」が現れます

## 使い方(ここからは全部シート上)

1. **✉️ Gmail整理 → ① 初期セットアップ** をクリック
   - 初回はGoogleの権限確認が出ます → 自分のアカウント選択 →「詳細」→「(安全でないページ)に移動」→ 許可
     (自分で貼り付けたスクリプトなので大丈夫です)
   - 「設定」シートが自動で作られます
2. **「設定」シートの黄色いセル**に、見逃したくない送信元やキーワードを書く
3. **② お試し実行** をクリック
   - メールは一切変更されず、「プレビュー」シートに **どのメールがどうなるか一覧表示** されます
4. プレビューに納得したら **③ 本番実行**
5. 気に入ったら **④ 毎日の自動整理を ON** — 以降は毎朝7時に勝手に整理されます

## 日々の運用

- 設定を変えたいとき → 「設定」シートのセルを書き換えるだけ(保存も再実行も不要、次回実行から反映)
- 何が整理されたか見たいとき → 「実行ログ」シート
- やめたいとき → メニューから「自動整理を OFF」

## 困ったら

- **メニューが出ない** → シートを再読み込み(F5)して数秒待つ
- **仕分けが効かない** → Gmail設定で受信トレイの「カテゴリ」タブ(プロモーション等)が有効か確認
- **もっとルールを増やしたい** → このフォルダの `Code.gs` を作った Claude に相談してください
